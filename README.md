# vue-ts-pinia-cms

基于Vue3.2、TypeScript、pinia、Vite、Element-Plus的后台管理解决方案



### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
